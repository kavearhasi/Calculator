# Calculator
A straightforward and efficient Calculator App for Android that performs basic arithmetic operations with a clean and user-friendly interface.


## Features

- **Addition (+)**
- **Subtraction (-)**
- **Multiplication (×)**
- **Division (÷)**
- Clean and intuitive user interface

## Tech Stack

- **Language:** Kotlin
- **UI:** XML Layouts
- **Development Environment:** Android Studio
